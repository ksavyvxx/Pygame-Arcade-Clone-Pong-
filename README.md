# 🎮 Игра Pong на Pygame

Классическая аркадная игра «Понг», созданная с чистого листа на базе библиотеки Pygame в рамках обучения в CapEducation.

## ✨ Особенности проекта
* **Динамическое ускорение мяча:** С каждым успешным ударом ракетки мяч летит быстрее, усложняя игру.
* **Режим с ИИ:** Возможность играть как вдвоём на одной клавиатуре, так и против умного ИИ бота.
* **Удобное переключение режимов:** Прямо во время игры кнопка `TAB` переключает противника.
## Управление:
1-Игрок:W-движение вверх S-движение вниз 
2-Игрок:⬆️-движение вверх ⬇️-движение вниз
